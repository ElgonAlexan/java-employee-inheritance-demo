# Java Employee Inheritance Demo

This project demonstrates **Object-Oriented Programming (OOP)** concepts in Java, focusing on **inheritance**, **constructor chaining**, **method overriding**, and **polymorphism**.

The system models different types of employees using a common superclass and specialized subclasses.

---

##  Concepts Covered

- Classes and Objects
- Inheritance (`extends`)
- Constructor chaining using `super()`
- Method overriding (`toString()`)
- Polymorphism
- Encapsulation (getters)
- Keyboard input using `Scanner`
